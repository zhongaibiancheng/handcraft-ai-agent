# 🤖 handcraft-ai-agent

**用一台普通电脑 + 免费工具，从零开始手搓一个真正能跑的 AI Agent。**

不花一分钱，不依赖任何付费 API。

## 🎯 这是什么？

一个从零开始学习 AI Agent 开发的实战教程系列。每节课对应一个核心概念，配合可直接运行的代码，循序渐进带你构建完整的 AI Agent。

> 💡 零门槛：只要会 Python 基础 + 会用命令行
> 💰 零成本：全部使用免费工具和本地模型
> 🚀 实战向：每课产出可运行的代码，不纸上谈兵

## 🚀 快速开始

### 1. 安装 Ollama

macOS / Linux / Windows 均可，官网下载：https://ollama.com/download

```bash
ollama --version
```

### 2. 下载模型

```bash
ollama pull qwen2.5:7b
```

### 3. 运行第一课代码

```bash
python ai_chat.py
```
